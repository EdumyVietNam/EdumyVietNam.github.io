# EdumyVietNam - Nền tảng học trực tuyến thế hệ mới

Website giới thiệu doanh nghiệp EdumyVietNam - EdTech startup với nền tảng học trực tuyến tích hợp AI.

## Công nghệ sử dụng

- HTML5 Semantic
- CSS3 (Glassmorphism, Gradient, Flexbox, Grid)
- Vanilla JavaScript ES6+
- Google Fonts (Inter)

## Tính năng

- Dark Mode / Light Mode
- Responsive Mobile First
- Glassmorphism & Gradient hiện đại
- Scroll Animation (Intersection Observer)
- Counter Animation
- Dark Mode Toggle (lưu localStorage)
- Loading Screen
- Back To Top Button
- FAQ Accordion
- Contact Form Validation
- Pricing Monthly/Yearly Toggle
- Course Filter
- Sticky Navigation + Mobile Menu

## Cấu trúc thư mục

```
├── index.html              # Trang chủ
├── css/
│   └── style.css           # Stylesheet chính
├── js/
│   └── main.js             # JavaScript chính
├── images/
│   └── logo.png            # Logo thương hiệu
├── assets/                 # Tài nguyên
└── pages/
    ├── about.html          # Giới thiệu
    ├── courses.html        # Khóa học
    ├── course-detail.html  # Chi tiết khóa học
    ├── exam.html           # Thi trực tuyến
    ├── ai.html             # Công nghệ AI
    ├── instructor.html     # Dành cho giảng viên
    ├── pricing.html        # Bảng giá
    ├── blog.html           # Blog
    ├── contact.html        # Liên hệ
    ├── faq.html            # Câu hỏi thường gặp
    └── policy.html         # Chính sách
```
