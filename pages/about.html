Đề tài khóa luận

Xây dựng nền tảng E-learning tích hợp quản lý khóa học và đánh giá trực tuyến

EDUMY là đề tài khóa luận tốt nghiệp của nhóm sinh viên Quang Nhật Hưng, Nguyễn Châu Kha và Anh Tuấn, dưới sự hướng dẫn của TS.Nguyễn Thị Bích Ngân.

Mục tiêu của đề tài là nghiên cứu, phân tích, thiết kế và xây dựng một nền tảng E-learning với kiến trúc Microservices, tích hợp liền mạch giữa quản lý khóa học và đánh giá trực tuyến, mang lại trải nghiệm tối ưu cho cả giảng viên và học viên.

Hệ thống gồm 5 module chính: Xác thực (JWT + RBAC), Quản lý hệ thống (Admin Dashboard + Workflow), Thanh toán (VNPAY QR), Quản lý khóa học (CRUD + Ngân hàng câu hỏi + Đánh giá) và AI (Chatbot RAG + Tự động sinh đề thi).

Kiến trúc hệ thống

5 module chức năng cốt lõi

EDUMY được thiết kế theo kiến trúc Microservices, chia làm 5 module độc lập, mỗi module đảm nhiệm một nhóm chức năng riêng biệt.

🔐

Module 1: Xác thực (Authentication)

Quản lý đăng ký (Email + OTP), đăng nhập Google/Microsoft SSO, cấp phát JWT và phân quyền RBAC cho 3 vai trò: Admin, Giảng viên, Học viên.

📊

Module 2: Quản lý hệ thống (System Management)

Quản lý người dùng tập trung, phê duyệt giảng viên qua Workflow, CRUD danh mục khóa học, Admin Dashboard với biểu đồ thống kê và nhật ký hoạt động.

💳

Module 3: Thanh toán (Payment)

Tích hợp cổng thanh toán VNPAY qua mã QR, quản lý giao dịch mua khóa học, đối soát doanh thu cho giảng viên và thực hiện yêu cầu rút tiền.

📖

Module 4: Quản lý khóa học (Course Management)

CRUD khóa học, cấu trúc học liệu Section → Lesson, ngân hàng câu hỏi 4 định dạng (Trắc nghiệm, Điền khuyết, Đúng/Sai, Tự luận), tạo đề thi và chấm điểm tự động hoặc thủ công.

🤖

Module 5: Trí tuệ nhân tạo (AI)

Chatbot hỏi đáp sử dụng kỹ thuật RAG để giải đáp thắc mắc theo ngữ cảnh tài liệu khóa học. Tự động sinh bài kiểm tra trắc nghiệm từ nội dung bài giảng (PDF, Slide, Transcript).

⚙

Kiến trúc Microservices

Backend .NET 8, Frontend React Vite & Flutter, cơ sở dữ liệu MySQL + Redis, API Gateway Nginx, đóng gói Docker, hỗ trợ Horizontal Scaling và CI/CD.
